# Rata-rata-Nilai #

# Deskripsi Program
Program RataRataNilai digunakan untuk menghitung rata-rata nilai ujian sejumlah siswa pada mata pelajaran Produktif RPL.
Program ini meminta pengguna untuk memasukkan jumlah siswa dan nilai masing-masing, lalu menghitung rata-ratanya secara otomatis.

# Cara Kerja Program
> Input Data dari Pengguna
- Program meminta pengguna untuk memasukkan jumlah siswa.
- Kemudian, pengguna diminta memasukkan nilai ujian setiap siswa satu per satu.
> Proses Perhitungan
- Program menggunakan perulangan for untuk menjumlahkan semua nilai siswa.
- Setiap nilai yang dimasukkan akan ditambahkan ke variabel totalNilai.
- Setelah semua nilai dimasukkan, rata-rata dihitung dengan rumus:
rataRata = totalNilai / jumlahSiswa
> Output Hasil
Program menampilkan hasil perhitungan berupa nilai rata-rata ujian seluruh siswa.

# Contoh Input dan Output
> Input:
Masukkan jumlah siswa: 3
Masukkan nilai siswa ke- 1: 80
Masukkan nilai siswa ke- 2: 90
Masukkan nilai siswa ke- 3: 70
> Output:
Rata-rata nilai ujian Produktif RPL adalah: 80.0

# Konsep yang Digunakan
> Input menggunakan Scanner untuk membaca data dari pengguna.
> Perulangan for untuk mengumpulkan nilai setiap siswa.
> Operasi aritmetika dasar untuk menghitung total dan rata-rata nilai.
> Variabel bertipe double untuk mendukung nilai desimal.
